SimPEG’s new inversion framework#

Welcome to the documentation of inversion-ideas: a prototype for a new inversion framework for SimPEG.

The main goal of this repository is to build a working prototype of a new inversion framework for SimPEG. Once the goal is achieved, the new framework will be ported to SimPEG, where it’ll be maintained and developed for the future.

Important

This repository will never host a stable and maintained codebase. Its sole purpose is to have a place to experiment with design ideas for the new framework.
